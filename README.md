---

## Project Structure

```
├── index.html          # Main portfolio (hero, about, projects, contact)
├── global.css          # Shared styles across all pages
└── blog/
    ├── index.html      # Blog listing page
    └── posts/
        ├── post-template.html   # Template for new posts
        └── *.html               # Individual blog posts
```

---

## Adding a Blog Post

1. Copy `blog/posts/post-template.html` and rename it (e.g. `my-post-title.html`)
2. Update the title, category, date, and content inside the file
3. Add a matching entry to the post list in `blog/index.html`
4. Commit and push — GitHub Pages deploys automatically
